API

Catalogue endpoint

When the backend is running, the furniture catalogue is available as a simple JSON API for any tool to consume.

Get products from a source

Returns a versioned list of products scraped from the given retailer.

GET http://localhost:3001/catalogue?source=west-elm

Response shape:

{
  "version": "...",
  "source": "west-elm",
  "builtAt": "...",
  "total": 0,
  "products": [ ... ]
}
Supported sources: rove-concepts · west-elm · soho-home · povison · claude-home · soho-home-lighting
